About Intelligent Tourist WebApp


Intelligent Tourist

This Webapp tells about the tourist system which helps the tourists to locate and find amenities in that particular city. A login page will appear for the users to connect with our webpage for the existing user and the registration form for the new users. The user has to fill the set of questionnaire which the user to reply using radio buttons. An intelligent tree algorithm will run and 1-3 best results will be provided from the online database. We also have provided option to view the exact location of the spot on Google map. Otherwise the user can also make an manual search for various tourists locations by country, city area, name of the place.

The users can also prefer a guide with their preffered language. And Finally after the user has the right to rate the guide so that it helps the Webapp for better recommendations in the future. Actually, this webapp saves their money and time in finding and consulting a travel agency where they charge more.
